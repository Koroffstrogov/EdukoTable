# EdukoTable — brief produit

## Vision

EdukoTable est une application de révision des multiplications pensée pour être ouverte souvent, utilisée rapidement, et perçue comme gratifiante par un enfant.

L’application doit être moins une “fiche d’exercices numérique” qu’une petite routine de progression :

```txt
Je choisis les tables que je veux travailler.
Je fais 10 questions rapides.
Je gagne des étoiles, stickers ou badges.
L’application retient mes difficultés sans me noyer dans les statistiques.
Je peux revenir demain.
```

## Public

- Enfant à partir d’environ 7 ans.
- Usage accompagné ou autonome.
- Priorité au tactile mobile.
- Parents intéressés par une progression simple, pas par un tableau d’analyse complexe.

## Nom

Nom final : **EdukoTable**

Ne pas utiliser d’autre nom dans l’interface, le manifest PWA, le titre HTML ou les textes visibles.

## Plateformes

- PWA React/Vite.
- Déploiement Vercel.
- Compatible iPhone Safari et Android Chrome.
- Fonctionnement local/offline autant que possible.
- Pas de compte.
- Pas de backend.
- Pas de publicité.

## Proposition de valeur

- Réviser les tables de 2 à 10.
- Sessions courtes.
- Quatre propositions à chaque question.
- Moteur simple mais intelligent.
- Récompenses attractives.
- Progression enregistrée localement.

## Principes de conception

### Simplicité

L’enfant ne doit pas configurer un exercice complexe. Il choisit une table ou un groupe de tables, puis démarre.

### Fluidité

Les questions doivent s’enchaîner vite. Les animations doivent renforcer le plaisir, pas ralentir la session.

### Gratification

Le système de récompense est une fonctionnalité centrale du MVP, pas un bonus cosmétique.

### Progression discrète

Les stats servent au moteur. Elles ne doivent pas dominer l’interface enfant.

### Bienveillance

Pas de vie perdue, pas de honte, pas d’écran d’échec agressif.

## Non-objectifs du MVP

- Pas de multi-compte.
- Pas de synchronisation cloud.
- Pas de classement.
- Pas de mode chronométré obligatoire.
- Pas de saisie clavier.
- Pas de boutique complexe.
- Pas de personnalisation poussée de l’avatar.
- Pas d’IA générative dans l’application.
